Soạn thảo văn bản



  1. NHẬP & SỬA DỮ LIỆU
    • Nhập dữ liệu thì khá dễ dàng, bạn chỉ cần chọn ô cần nhập dữ liệu là xong.
    • Còn sửa dữ liệu thì bạn cần kích đúp chuột hoặc nhấn F2
  2. CÁC KIỂU DỮ LIỆU EXCEL
    • Văn bản
    • Số
    • Ngày tháng
    • Tiền tệ
    • Tỉ lệ phần trăm
      • ...
  3. SAO CHÉP, DÁN DỮ LIỆU
    • Ctrl + C: Copy dữ liệu
    • Ctrl + V: Paste dữ liệu
    • Phím tắt tự tạo, ví dụ: Alt + 1 dùng để paste special value -> Bạn sẽ được học trong các khóa nâng cao.
    • Khi dùng chuột bạn hãy click chuột phải vào vùng cần copy rồi chọn copy là được.
