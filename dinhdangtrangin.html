Định dạng trang in



  1. 3 CHẾ ĐỘ XEM TRANG
    • Normal view
    • Page break preview
    • Page Layout
  2. TRANG IN vs. VÙNG IN
    • Tạo vùng in giúp bạn tiết kiệm nhiều thời gian căn chỉnh và cũng giúp tiết kiệm giấy in khi.
    • Tạo vùng in = Set print area
  3. CHỌN VÙNG IN NHƯ THẾ NÀO
    • Thẻ Page Layout
    • Chọn print area
      • Và tiến hành chọn vùng để set print area
